1: MongoDB Atlas (Cloud - Recommended)
Go to MongoDB Atlas
Create free account
Create a new cluster
Get connection string (looks like: mongodb+srv://username:password@cluster.mongodb.net/)

2: Redis Cloud (Recommended)
Go to Redis Cloud
Create free account
Create database
Get connection string

3: ENV FILE :-

# Server Configuration
PORT=5000
NODE_ENV=development

# Database - Replace with your actual connection strings
MONGODB_URI=mongodb://localhost:27017/ayurchakra
# OR for MongoDB Atlas:
# MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/ayurchakra

REDIS_URL=redis://localhost:6379
# OR for Redis Cloud:
# REDIS_URL=redis://username:password@host:port

# JWT Configuration
JWT_SECRET=your-super-secret-jwt-key-here-make-it-long-and-random
JWT_EXPIRE=7d

# Blockchain Configuration (Optional for now)
ETHEREUM_RPC_URL=https://sepolia.infura.io/v3/your-project-id
